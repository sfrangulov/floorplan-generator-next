# Анализ планировки: 3-комнатная квартира, 72.8 м²

## 1. Идентификация помещений

На плане идентифицированы следующие помещения:

| # | Помещение | Ориентировочная площадь | Окна | Балкон |
|---|---|---|---|---|
| 1 | **Кухня-гостиная** (верхний левый угол) | ~22–25 м² | Да (2+ окна + балконная дверь) | Да (лоджия) |
| 2 | **Детская** (верхний центр) | ~10–12 м² | Да (1 окно) | Нет |
| 3 | **Спальня 1** (верхний правый угол) | ~12–14 м² | Да (1 окно) | Нет |
| 4 | **Спальня 2 (мастер)** (нижний левый угол) | ~14–16 м² | Да (балконная дверь) | Да (лоджия) |
| 5 | **Санузел 1** (совмещённый — ванна) | ~3,5–4 м² | Нет | — |
| 6 | **Санузел 2** (туалет/душевая) | ~2,5–3 м² | Нет | — |
| 7 | **Прихожая / коридор** | ~5–7 м² | Нет | — |

**Класс жилья:** Комфорт (72.8 м² для 3-комн. попадает в диапазон 65–90 м²)

## 2. Соответствие правилам планировки (Документ 1)

### 2.1 Площади — СООТВЕТСТВУЕТ

| Правило | Норма | Факт | Статус |
|---|---|---|---|
| Общая площадь 3-комн. (муниципальный фонд) | 56–65 м² | 72.8 м² | ✅ Превышает минимум |
| Площадь общей комнаты (2+ комн.) | ≥ 16 м² | ~22–25 м² | ✅ |
| Площадь спальни (2 чел.) | ≥ 10 м² | ~12–16 м² | ✅ |
| Площадь детской (1 чел.) | ≥ 8 м² | ~10–12 м² | ✅ |
| Кухонная зона в кухне-столовой | ≥ 6 м² | ~6–8 м² | ✅ |

### 2.2 Состав помещений — СООТВЕТСТВУЕТ

| Обязательный элемент | Статус |
|---|---|
| Жилая комната (3 шт.) | ✅ Кухня-гостиная + спальня + детская (или 2 спальни + детская) |
| Кухня / кухня-столовая | ✅ Кухня-гостиная |
| Прихожая | ✅ Есть |
| Ванная / душевая | ✅ Санузел 1 |
| Уборная / совмещённый санузел | ✅ Санузел 2 |
| Кладовая / хозяйственный шкаф | ⚠️ Не видно на плане — возможен дефицит |

### 2.3 Окна и инсоляция — СООТВЕТСТВУЕТ

| Правило | Статус | Комментарий |
|---|---|---|
| Окна во всех жилых комнатах | ✅ | Все 3 жилые комнаты + кухня-гостиная имеют окна |
| Площадь остекления ≥ 1/8 площади пола | ✅* | Визуально окна достаточного размера (*точно без размеров не проверить) |
| Инсоляция ≥ 1 комнаты | ✅* | Зависит от ориентации здания |

### 2.4 Проходимость комнат — СООТВЕТСТВУЕТ

| Правило | Статус | Комментарий |
|---|---|---|
| Спальни — непроходные | ✅ | Все 3 комнаты имеют отдельные входы из коридора |
| Кухня-гостиная допускает проходной вариант | ✅ | Через неё не осуществляется транзит в спальни |

### 2.5 Двери — ПРЕИМУЩЕСТВЕННО СООТВЕТСТВУЕТ

| Правило | Статус | Комментарий |
|---|---|---|
| Входная дверь ≥ 800 мм | ✅ | Стандартная входная дверь |
| Двери санузлов открываются наружу | ✅* | Визуально — наружу в коридор |
| Двери не сталкиваются при открывании | ⚠️ | В зоне коридора, при большом количестве дверей, **возможно столкновение** — требует уточнения размеров |

### 2.6 Функциональное зонирование — ХОРОШО

| Критерий | Оценка |
|---|---|
| Разделение дневной/ночной зон | ✅ Кухня-гостиная — дневная зона; спальни и детская — ночная |
| Транзит через ночную зону | ✅ Отсутствует — хорошо |
| Мокрые зоны сгруппированы | ✅ Оба санузла расположены рядом (один блок стояков) |
| Вход → прихожая → развилка | ✅ Из прихожей есть доступ ко всем зонам |

### 2.7 Смежность помещений — СООТВЕТСТВУЕТ

| Правило | Статус | Комментарий |
|---|---|---|
| Вход в санузлы из коридора/холла | ✅ | Оба санузла имеют вход из коридора |
| Запрет санузла над жилыми соседей | — | Зависит от расположения в здании (не нарушено при типовой секции) |
| Кухня не через спальню | ✅ | Кухня-гостиная доступна из коридора напрямую |

## 3. Соответствие правилам расстановки (Документ 2)

### 3.1 Кухонная зона

| Правило | Норма | Оценка | Статус |

| Рабочий треугольник (периметр) | 4–6 м (опт.) | Линейная планировка — треугольник вырожденный | ⚠️ Допустимо, но не оптимально |
| Длина линейной кухни | ≤ 3660 мм | Визуально в пределах нормы | ✅* |
| Плита — боковая стена | ≥ 200 мм | Не определить точно | ✅* |
| Плита — окно | ≥ 450 мм | На плане плита расположена вдоль стены, не у окна | ✅ |
| Вытяжка — плита | 650–750 мм (электро) / 750–850 мм (газ) | Не видно вытяжку | — |
| Проход перед кухней | ≥ 900 мм | Визуально достаточно (открытое пространство) | ✅ |

## 4. Выявленные замечания

### КРИТИЧЕСКИЕ (нарушение норм)

Критических нарушений не обнаружено.

### СУЩЕСТВЕННЫЕ (требуют внимания)

| # | Замечание | Правило | Рекомендация |
|---|---|---|---|
| 1 | **Газовая плита в кухне-гостиной** | СП 54.13330: кухня с газовой плитой должна быть отделена дверью от жилых помещений | Если плита газовая — необходима раздвижная перегородка или замена на электрическую |
| 2 | **Отсутствие кладовой / хоз. шкафа** | СП 54.13330, п. 5.3: кладовая обязательна | Предусмотреть встроенный шкаф в коридоре или нишу |
| 3 | **Столкновение дверей в коридоре** | Двери не должны сталкиваться при открывании | Проверить радиусы открывания; при необходимости — раздвижные двери |
| 4 | **Теснота в совмещённом санузле** | Зона выхода из ванны ≥ 550 мм; перед раковиной ≥ 700 мм | Проверить точные размеры; возможна замена ванны на душевую кабину |

### РЕКОМЕНДАТЕЛЬНЫЕ (улучшения)

| # | Замечание | Рекомендация |
|---|---|---|
| 5 | Линейная кухня — рабочий треугольник вырожденный | При площади кухонной зоны 6–8 м² — допустимо, но Г-образная была бы удобнее |
| 6 | Мастер-спальня плотно меблирована (шкафы по стенам) | Проверить проход ≥ 700 мм вокруг кровати с 2+ сторон |
| 7 | Одно окно в детской | При площади 10–12 м² одного окна достаточно, но проверить соотношение 1:8 |
| 8 | Нет гардеробной | Для класса «Комфорт» желательна хотя бы гардеробная ниша |
| 9 | Второй санузел — проверить наличие ванны/душа | Если это только туалет, то для 3-комн. квартиры класса «Комфорт» рекомендуется два полноценных санузла |
| 10 | Расстояние от обеденного стола до ближайшей мебели (дивана) | Проверить ≥ 900 мм для прохода за стулом |

### Резюме

Планировка **в целом грамотная** и соответствует основным нормам СП 54.13330 для 3-комнатной квартиры класса «Комфорт». Сильные стороны — хорошее функциональное зонирование, два санузла, два балкона, все спальни непроходные. Основные зоны риска: возможная необходимость перегородки при газовой плите в кухне-гостиной, отсутствие выделенного хранения (кладовая/гардеробная), и потенциальная теснота в санузле и коридорной зоне с дверями.
