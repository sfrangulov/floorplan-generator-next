# Правила планировки квартир в многоквартирных жилых домах

## Нормативная база

Основные документы, регулирующие проектирование квартир в России:

- **СП 54.13330.2022** — актуализированная редакция СНиП 31-01-2003 «Здания жилые многоквартирные»
- **СНиП 31-01-2003** — базовый документ по жилым многоквартирным зданиям
- **СанПиН 2.2.1/2.1.1.1076-01** — гигиенические требования к инсоляции
- **ГОСТ 31173-2016** — блоки дверные стальные
- **ГОСТ 6629-88** — блоки дверные деревянные
- **СНиП 23-05-95** — естественное и искусственное освещение

## 1. Классификация помещений квартиры

### 1.1 Жилые помещения

| Тип помещения | Описание | Требует окна |
|---|---|---|
| Гостиная (общая комната) | Основное жилое помещение для дневного пребывания | Да |
| Спальня | Помещение для сна и отдыха | Да |
| Детская | Жилая комната для детей | Да |
| Кабинет | Жилая комната для работы | Да |

### 1.2 Вспомогательные (подсобные) помещения

| Тип помещения | Описание | Требует окна |
|---|---|---|
| Кухня | Приготовление пищи, мойка, плита | Да |
| Кухня-столовая | Кухня + обеденная зона | Да |
| Кухня-ниша | Нишевая кухня (в студиях) | Нет (при электроплите) |
| Прихожая (передняя) | Входная зона | Нет |
| Коридор | Связующее помещение | Нет |
| Холл | Расширенное связующее помещение | Нет |
| Ванная комната | Ванна/душ + умывальник | Нет |
| Туалет (уборная) | Унитаз | Нет |
| Совмещённый санузел | Ванна/душ + умывальник + унитаз | Нет |
| Кладовая | Хранение | Нет |
| Гардеробная | Хранение одежды | Нет |
| Постирочная | Стиральная/сушильная машина | Нет |
| Балкон / лоджия | Открытое/полуоткрытое пространство | Не применимо |

### 1.3 Классификация по «мокрым зонам»

- **Мокрые зоны** — помещения с водоснабжением и гидроизоляцией: санузлы, ванные, душевые, постирочные, кухни
- **Сухие зоны** — жилые комнаты, коридоры, кладовые, гардеробные

## 5. Правила стыковки (смежности) помещений

### 5.1 Обязательные связи

| Из какого помещения | Куда должен быть доступ | Комментарий |
|---|---|---|
| Входная дверь | → Прихожая/холл | Обязательно через прихожую |
| Прихожая | → Коридор / общая комната | Связь с остальной квартирой |
| Кухня | → Коридор / столовая | Не допускается вход только через спальню |
| Санузел (основной) | → Коридор / холл | Стандартный вход |
| Санузел (при спальне) | → Спальня | Только при наличии второго санузла с входом из коридора |

### 5.2 Запрещённые связи

| Правило | Основание |
|---|---|
| Вход в санузел с унитазом непосредственно из кухни | СП 54.13330 п. 5.12 |
| Вход в единственный санузел из жилой комнаты (без второго санузла из коридора) | СП 54.13330 |
| Проходная спальня (в 2+ комн. квартирах муниципального фонда) | СНиП 31-01-2003 |
| Жилые помещения в подвальном/цокольном этаже | СП 54.13330 |

### 5.3 Матрица допустимой смежности

Обозначения: **+** допускается, **−** запрещено, **(у)** — условно (при выполнении условий)

| | Прихожая | Коридор | Гостиная | Спальня | Кухня | Ванная | Туалет | С/у совм. | Кладовая |
|---|---|---|---|---|---|---|---|---|---|
| **Прихожая** | — | + | + | + | + | + | + | + | + |
| **Коридор** | + | — | + | + | + | + | + | + | + |
| **Гостиная** | + | + | — | + | + | (у) | − | (у) | + |
| **Спальня** | + | + | + | — | − | (у) | − | (у) | + |
| **Кухня** | + | + | + | − | — | − | − | − | + |
| **Ванная** | + | + | (у) | (у) | − | — | — | — | − |
| **Туалет** | + | + | − | − | − | — | — | — | − |
| **С/у совм.** | + | + | (у) | (у) | − | — | — | — | − |
| **Кладовая** | + | + | + | + | + | − | − | − | — |

**(у) — условно допускается:** Вход из спальни в ванную/совмещённый санузел допускается при наличии второго туалета с входом из коридора.

### 5.4 Правила расположения мокрых зон

1. **Мокрые зоны не должны располагаться над жилыми комнатами и кухнями соседей** (вертикальное соответствие)
2. Расширение мокрых зон допускается только за счёт нежилых помещений (коридор, кладовая)
3. Расширение мокрых зон за счёт кухни — запрещено
4. **Исключение:** На первом этаже или над нежилыми помещениями ограничений нет
5. Гидроизоляция пола в мокрых зонах — обязательна
6. Порожек на входе в мокрую зону — не менее 15–20 мм (или уровень пола ниже смежного помещения)

### 5.5 Принцип функционального зонирования

Квартира делится на три функциональные зоны:

1. **Входная зона:** прихожая, холл, гардеробная при входе
2. **Дневная зона (общественная):** гостиная, кухня, столовая, гостевой санузел
3. **Ночная зона (приватная):** спальни, детские, ванные при спальнях, кабинет

**Правило:** Дневная и ночная зоны не должны пересекаться транзитным движением. Идеальная схема: вход → прихожая → развилка на дневную и ночную зоны.

### 5.6 Центральное положение общей комнаты

Общая жилая комната (гостиная) по своему назначению требует **центрального положения** в квартире — она является связующим звеном между входной зоной (прихожая/холл) и остальными помещениями. В квартирах комфорт-класса и выше гостиная может быть проходной при условии, что из неё обеспечивается проход не более чем в 2 смежные комнаты.

### 5.7 Правила размещения обеденной зоны в гостиной-столовой

- Обеденная группа (стол со стульями) размещается **ближе к кухне** для удобства сервировки
- Обеденная группа **не ставится напротив входа** в комнату — сидящим спиной к дверному проёму некомфортно
- Место спиной ко входу в помещение заполняется последним

## 6. Правила расположения дверей

### 6.1 Размеры дверных проёмов

| Тип двери | Ширина полотна, мм | Высота полотна, мм | Ширина проёма, мм |
|---|---|---|---|
| Входная дверь | 860–960 | 2050 | 900–1010 |
| Межкомнатная (жилая комната) | 800 | 2000 | 870 |
| Межкомнатная (гостиная) | 800–900 | 2000 | 870–970 |
| Двустворчатая (гостиная) | 1300–1500 | 2000 | 1370–1570 |
| Кухня | 700–800 | 2000 | 770–870 |
| Ванная / туалет | 600 | 1900–2000 | 685 |
| Совмещённый санузел | 600–700 | 1900–2000 | 685–770 |

### 6.2 Правила размещения дверей

1. **Минимальное расстояние от двери до смежной стены:** 100 мм (для размещения наличника и удобства открывания)
2. **Направление открывания:**
   - Входная дверь — наружу (в сторону эвакуации)
   - Двери санузлов — наружу (в коридор) для безопасности
   - Межкомнатные двери — в сторону большего помещения или по направлению естественного движения
3. **Запрет на столкновение полотен:** двери, расположенные напротив или рядом, не должны сталкиваться при одновременном открывании
4. **Свободное пространство перед дверью:** не менее ширины дверного полотна
5. **Двери не должны перекрывать проход** в коридоре при открывании
6. **Расстояние между дверями на одной стене:** минимум 100–150 мм

### 6.3 Противопожарные требования

- Минимальная ширина входной двери: 800 мм (для эвакуации, СП 3.13130.2009)
- Входная дверь должна открываться наружу и не блокировать общий коридор

## 7. Правила расположения окон

### 7.1 Обязательность окон

| Помещение | Окно обязательно | Примечание |
|---|---|---|
| Жилые комнаты | Да | Всегда |
| Кухня | Да | Кроме кухни-ниши с электроплитой |
| Кухня-столовая | Да | — |
| Кухня-ниша | Нет | Только с электроплитой и вытяжкой |
| Санузлы | Нет | Вентиляция обязательна |
| Коридоры, прихожие | Нет | — |
| Кладовые, гардеробные | Нет | — |

### 7.2 Минимальная площадь остекления

- **Отношение площади окна к площади пола:** 1:8 — 1:5,5 (10–18% площади пола)
- Рекомендуемое: 1:6 (≈ 16,7% площади пола)
- Для комнаты 20 м² — окно не менее 2,5 м² светового проёма

### 7.3 Инсоляция (требования СанПиН)

| Параметр | Значение |
|---|---|
| Минимальная продолжительность (центральная зона, 48–58° с.ш.) | 2 часа |
| Минимальная продолжительность (северная зона) | 2,5 часа |
| Минимальная продолжительность (южная зона) | 1,5 часа |
| Расчётный период | 22 марта — 22 сентября |
| Минимум инсолируемых комнат в 1–3-комн. квартире | не менее 1 |
| Минимум инсолируемых комнат в 4+ комн. квартире | не менее 2 |

### 7.4 Расположение окон

1. Окна жилых комнат предпочтительно ориентировать на юг, юго-восток, юго-запад
2. Кухни допускается ориентировать на север
3. Для обеспечения инсоляции учитываются окна в секторе 135°–225° по азимуту
4. Окно должно быть расположено так, чтобы расстояние от него до наиболее удалённой стены не превышало 6 м (глубина комнаты при одностороннем освещении)
5. Высота подоконника: 700–900 мм от уровня пола (стандарт 800 мм)
6. Расстояние от окна до боковой стены: не менее 250 мм

## 8. Инженерные коммуникации — правила привязки

### 8.1 Стояки водоснабжения и канализации

- Стояки размещаются в мокрых зонах (санузлы, кухня)
- Мокрые зоны группируются вертикально (этаж над этажом) для оптимизации стояков
- Кухня и санузел рекомендуется размещать смежно или через стену для минимизации разводки
- Максимальное удаление сантехприбора от стояка канализации: 3 м (для обеспечения уклона)

### 8.2 Вентиляция

- Вытяжные каналы обязательны в кухне, ванной, туалете, совмещённом санузле
- Вентканалы выводятся в капитальных (несущих) стенах или через шахты
- Кухня с газовой плитой — обязательно окно + вытяжной канал
- Кухня-ниша — только электроплита + механическая вытяжка

### 8.3 Электроснабжение

- Электрощиток: в прихожей или коридоре
- Распределительные коробки: под потолком, в местах разветвления
- Розетки: не ближе 0,5 м от газовых труб, не ближе 0,6 м от края ванны/душа

## 10. Правила проходимости (непроходные комнаты)

1. В квартирах с 2+ комнатами для государственного/муниципального фонда: **спальни должны быть непроходными**
2. Общая комната (гостиная) может быть проходной
3. Кухня не должна быть единственным проходом в жилую комнату
4. Допускается проходная гостиная при условии, что из неё обеспечивается проход не более чем в 2 смежные комнаты
5. Вход в кухню с газовой плитой не должен быть через жилую комнату

## 12. Типовые схемы стыковки помещений

### Схема 1: Компактная (эконом, 1–2 комнаты)

```
[Вход] → [Прихожая] → [Коридор] → [Гостиная]
                          ↓              ↓
                      [Санузел]     [Кухня]
                          ↓
                      [Спальня] (для 2-комн.)
```

### Схема 2: Линейная (комфорт, 2–3 комнаты)

```
[Вход] → [Холл] → [Коридор] ──→ [Спальня 1]
                       ↓    ──→ [Спальня 2]
                   [Гостиная] ←→ [Кухня-столовая]
                       ↓
                   [С/у гостевой]
                   [С/у при спальне]
```

### Схема 3: Зональная (бизнес/премиум, 3+ комнаты)

```
[Вход] → [Холл] ──→ Дневная зона: [Гостиная] ←→ [Кухня-столовая]
              │                         ↓
              │                   [Гост. санузел]
              │
              └──→ Ночная зона: [Коридор] → [Спальня 1 + ванная]
                                    ↓
                              [Спальня 2 + ванная]
                                    ↓
                              [Детская]
                                    ↓
                              [Кабинет]
```
